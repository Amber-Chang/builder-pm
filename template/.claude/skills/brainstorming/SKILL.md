# Brainstorming（薄紀律版）

<!-- 種子輕薄版 by builder-pm。要完整版可換成外部 brainstorming skill。 -->

## 心法
- **一次問一個問題**,別一次轟一堆。
- 先搞懂：目的 / 限制 / 成功長怎樣。
- 提 2-3 個做法 + 取捨 + 你的建議,讓 PM 選。
- **動手前先把設計講出來、拿到同意**,再進實作（對應憲章 #3）。

## 產出
收斂成：**Goal**（完成後的具體影響）+ **Out of scope**（這次不做）+ 選定做法。
